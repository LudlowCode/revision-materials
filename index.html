'Parsons problems'

This website is where you rearrange the pseudocode or code for the key algos that you need for your A-level CompSci. Follow it up with trying to code them yourself, or apply them on a dummy set of data. It's a must to prep for module 2.

Programming practice and similar

  • Python-practice.com is lovely and simple to get into. Great practice for exam questions. Do this first.
  • sql-practice.com is another simple one to check your skills.
  • Codingbat.com is for levelling up in Python and Java. Simple but properly free with no gimmicks.
  • Leetcode.com is great to challenge yourself and to prepare for interviews etc. It forces you down the OOP route, which seems a bit weird for python sometimes. It also encourages codegolf and overoptimisation, leading to a lot of horrible answers!
  • Codewars.com is gimmicky in the extreme but challenging, addictive and with an active community. Takes a whil to get into it.
